# Assignment 1 - Causal Inference

This folder contains the first assignment of the Causal Inference course.

The goal of this analysis is to introduce the main concepts of causal inference through exploratory data analysis and the comparison between treatment and control groups.

The analysis includes:

- loading and preprocessing the dataset;
- descriptive statistics of the main variables;
- comparison between treated and untreated units;
- visualization of the main characteristics of the sample;
- evaluation of differences between groups before estimating causal effects.

The purpose of this assignment is to understand the structure of the data and verify the assumptions required for causal analysis.

The analysis is implemented in **R** using statistical and visualization packages.
